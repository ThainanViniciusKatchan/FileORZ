Versão 1.3

Português-BR

### Versão de lançamento
- Primeira versão lançada, contendo as seguintes funcionalidades:
    - Organização de arquivos
    - Configuração de intervalo de tempo de verificação
    - Configuração de pasta para organização
    - Configuração de quais formatos serão organizados

### Bugs conhecidos
- Ao iniciar o aplicativo ele pode fechar sozinho instantaneamente
- Ao desativar a inicialização do aplicativo ele pode exibir uma mensagem de erro
- Caso você não selecione a pasta de organização ele pode inicar o script e dar erro por conta de não ter pasta selecionada

### Melhorias futuras
- Adicionar a opção de selecionar multiplas pastas para organização
- Adicionar a opção de personalizar o nome das pastas da organização
- Adicionar a opção de personalizar os formatos que serão organizados (Adicionando os formatos necessários manualmente)
- Compilar o aplicativo para Linux

Inglês-US

### Version release
- First version released, containing the following features:
    - File organization
    - Verification interval configuration
    - Folder configuration for organization
    - Configuration of which formats will be organized

### Known bugs
- When starting the application it can close instantaneously
- When disabling the application startup it can display an error message
- If you do not select the organization folder it can start the script and give an error because there is no folder selected

### Future improvements
- Add the option to select multiple folders for organization
- Add the option to customize the organization folder names
- Add the option to customize the formats that will be organized (Adding the formats manually)
- Compile the application for Linux
